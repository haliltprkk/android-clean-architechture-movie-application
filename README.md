# Clean Architecture Movie Application

Simple android movie application to practice clean architecture and some popular
methodologies/technologies.

## Tech Stack

- Hilt
- Coroutine Flow
- Unit Testing
- Google Truth
- Mockito
- Retrofit 2 / Okhttp3
- View Binding
- ktlint
